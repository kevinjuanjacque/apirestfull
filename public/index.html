Api para asignaturas
by Kevin Jacque AND Felipe Ramirez

Trabajo para TC

Se creo una API en donde se pueden guardar las asignaturas, con su fecha hora de comienzo hora de fin que comienza cada clase , también se pueden agregar tareas para cada asignatura y poner su fecha límite de entrega, en resumen una API para organizar las clases y trabajos



REQUEST ASIGNATURAS

Sobre la API, como informacion general todo las request son en formato json.
Crear asignatura
Para agregar una asignatura se debe crear un request, estilo "POST" en donde, debe indicarse nombre de asignatura, cantidad de bloques, hora de fin y hora de inicio en caso de que sean más de un bloque, la hora debe separarce por una coma.

Ver asignatura

Si deseas revisar las asignaturas se debe realizar una petición de estilo "GET", sin ningún parámetro de esta forma se devolverá las asignaturas agregadas y en caso contrario un arreglo vacío.

Eliminar asignatura

En este caso se debe utilizar una petición "DELETE", además de mandar por parámetro el id de la asignatura correspondiente que se desea eliminar.

Buscar una asignatura específica

En caso de querer encontrar una asignatura en específico, se debe hacer una petición, de estilo "GET", y enviar por parámetro el id de la asignatura.

REQUEST TAREAS

Sobre la API, como informacion general todo las request son en formato json.
Crear TAREA
Para agregar una tarea se debe crear un request, estilo "POST" en donde, debe indicarse ID de la asignatura perteneciente, Título, Fecha de entrega, y si es en grupos o no.

Ver tareas

Al revisar las tareas se debe realizar una petición de estilo "GET", sin ningún parámetro, de esta forma se devolverá todas las tareas agregadas y en caso contrario un arreglo vacío.

Eliminar tarea

En este caso se debe utilizar una petición "DELETE", además se debe enviar por parámetro el id de la tarea correspondiente que se desea eliminar.
